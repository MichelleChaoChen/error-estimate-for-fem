Convergence Experiments¶

convergence_experiments.convergence_experiments(tolerance, max_iter, file_nn, file_explicit, file_recovery)¶

Verifies the convergence AMR with neural network and compares performance with a recovery-based estimator.

Parameters:

  • tolerance – Global error tolerance

  • max_iter – Maximum number of iterations of refinement

convergence_experiments.convergence_plots(nn_file, exp_file, rec_file)¶

Creates several plots that demonstrate the convergence of adaptive mesh refinement using neural network as error estimator and compares the performance with recovery-based estimator: 1. Comparison of the MSE in error estimation of neural network and recovery method 2. Comparison of the average mesh size over iterations 3. Comparsion of the average number of iterations refinement needed

Parameters:

  • nn_file – File containing AMR data with neural network

  • exp_file – File containing AMR data with explicit error estimator

  • rec_file – File containing AMR data with recovery-based error estimator
