Neural Network Training & Tuning¶

class train.NeuralNetwork(dataset, epochs)¶
__init__(dataset, epochs)¶

Initializes the neural network with data and a tuner.

Parameters:

  • dataset – Dataset for training and testing

  • epochs – The number of epochs for which it should be trained

__weakref__¶

list of weak references to the object (if defined)

build_model(hp)¶

Builds a model for tuning hyperparameters and for training if the best hyperparameters have already been determined.

Parameters:

hp – Hyperparameters

Returns:

Neural network

custom_loss(y_true, y_pred)¶

Used for constructing/testing different loss functions.

Parameters:

  • y_true – True labels

  • y_pred – Predicted labels

Returns:

Loss of current predictions

evaluate()¶

Evaluates the performance of the neural network on testing data.

plot_training_loss(history)¶

Plots how the loss evolves as the model is trained. This is not called by default, add the method call to the constructor if a visualisation of training is needed.

Parameters:

history – Training history

save(name)¶

Saves the model in h5 format in the models directory.

Parameters:

name – The name the model should be saved as

train(best_hps, epochs=100)¶

Trains the model using the best parameters.

Parameters:

  • best_hps – Best hyperparameters

  • epochs – Number of epochs to train for, defaults to 100

Returns:

Trained neural network

tune_model()¶

Tunes the neural network by searching for the best hyperparameters.

Returns:

The best hyperparameters found
