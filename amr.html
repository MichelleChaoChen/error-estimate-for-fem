Adaptive Mesh Refinement¶

Adaptive Mesh Refinement (AMR) pipeline methods.

adaptive_mesh_refinement.adaptive_mesh_refinement(tolerance, max_iter, bc, source_func, error_estimator, mesh)¶

Performs Adaptive Mesh Refinement: 1. Initializes parameters of the problem 2. Computes a base solution on a coarse mesh 3. Refines mesh until the global error falls below desired threshold 4. Returns final solution

Parameters:

  • tolerance – Global error tolerance

  • max_iter – Maximum iterations of refinement

  • bc – Boundary condition of problem

  • source_func – Array of values describing the source function

  • error_estimator – User-selected error estimator

Returns:

Data from running the AMR pipeline

adaptive_mesh_refinement.build_nn_error_estimator(bc, source_func_str, nn_fine, nn_coarse, mesh_coarse)¶

Builds an error estimate function with currying so that there is an uniform interface for getting the error estimate for different estimators (e.g. recovery-based estimators).

Parameters:

  • bc – Boundary condition

  • source_func_str – Source function in string format (for FEniCS)

  • nn_fine – Neural network for fine elements

  • nn_coarse – Neural network for coarse elements

Returns:

A error estimate function

adaptive_mesh_refinement.extend_boundary(sol, grid, step)¶

Extends the boundaries on a mesh using virtual points.

Parameters:

  • sol – Computed solution

  • grid – The grid to extend the boudary on

  • step – Step sizes of the grid

Returns:

Solution, grid, and stepsize with virtual points on the boundary

adaptive_mesh_refinement.extend_boundary_rotate(sol, grid, step)¶

Extends the boundaries on a mesh using virtual points. Prevents the jump at the boundary element being equal to 0.

Parameters:

  • sol – Computed solution

  • grid – The grid to extend the boudary on

  • step – Step sizes of the grid

Returns:

Solution, grid, and stepsize with virtual points on the boundary

adaptive_mesh_refinement.generate_data(new_sol, new_grid, old_sol, old_grid)¶

Creates feature vectors for all elements of new FEM solution using both the new solution and base solution.

Parameters:

  • new_sol – New FEM solution

  • new_grid – Mesh on which the new FEM solution was computed

  • old_sol – Base FEM solution

  • old_grid – Coarse mesh for base solution

Returns:

Feature vector used to predict error of new FEM solution

adaptive_mesh_refinement.get_error_estimate(features, nn_fine, nn_coarse)¶

Provides the estimate of the local error and global error of feature vectors from one iteration of adaptive mesh refinement.

Parameters:

  • features – Feature vectors for every element on current mesh

  • nn_fine – Neural network for fine elements

  • nn_coarse – Neural network for coarse elements

Returns:

Array of local error on each element and global error

adaptive_mesh_refinement.refine(mesh, err_pred, global_error)¶

Refines mesh based on local error estimate: - An element is split into 2 equal halves if local error is high - Two elements are merged if the local error is too low

Parameters:

  • mesh – Mesh to refine

  • err_pred – Local error on every element

  • global_error – Global error threshold

Returns:

Mesh refined based on local error

adaptive_mesh_refinement.relative_change(grad_new, grad_old)¶

Computes relative change between a batch of quantities.

Parameters:

  • grad_new – Quantities from most recent FEM solution

  • grad_old – Quantities from base FEM solution

Returns:

The relative change in gradient for a patch of elements

adaptive_mesh_refinement.run_adaptive_mesh_refinement(tolerance, max_iter)¶

Wrapper around the adaptive_mesh_refinement() method so that the error estimator can be passed as parameter.

Parameters:

  • tolerance – Global error tolerance

  • max_iter – Maximum iterations of refinement

Returns:

FEM solution at each iteration
