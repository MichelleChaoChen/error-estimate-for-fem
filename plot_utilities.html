Plotting Utilities¶

The plotting utilities presented can be used to visualise the convergence and performance of our error estimator

plot_utilities.compute_average_elements(data)¶

Computes the average mesh size over multiple AMR runs for different source functions.

Parameters:

data – Data from multiple AMR runs

Returns:

THe average mesh size over the runs

plot_utilities.compute_mse_error_estimate(data)¶

Computes the MSE in error estimation, i.e. how far away is the estimated error from the exact error.

Parameters:

data – Data from multiple AMR runs

Returns:

MSE of error estimation over numerous iterations

plot_utilities.plot_average_iterations(avg_run_nn, avg_run_rec)¶

Plots the average number of iterations needed by neural network estimator and recovery-based estimator in AMR for the global error to fall below the desired threshold.

Parameters:

  • avg_run_nn – Average number of runs with neural network estimator

  • avg_run_rec – Average number of runs with recovery-based estimator

plot_utilities.plot_mse_error_estimate(data_nn, data_exp, data_rec)¶

Plots the error (MSE) of the neural network error estimator and the recovery-based error estimator. Provides a visual comparison of the two methods. The results can be found in the experiments directory.

Parameters:

  • data_nn – AMR data with neural network error estimator

  • data_exp – AMR data with explicit error estimator

  • data_rec – AMR data with recovery-based error estimator

plot_utilities.plot_number_elements(data_nn, data_rec)¶

Plots how the average mesh size changes as the number of iterations in AMR increases. This offers a comparison between mesh size with the neural network and recovery method.

Parameters:

  • data_nn – AMR data with neural network error estimator

  • data_rec – AMR data with recovery-based error estimator

plot_utilities.plot_refinement(x, exact, meshes, solutions, est_global_errors, ex_global_errors, N_elements, plot_title, color=True)¶

Plots how the mesh is refined and shows the global error estimates over iterations of refinement before the global error falls below the desired threshold.

Parameters:

  • x – The domain

  • exact – The exact solution

  • meshes – Meshes of each iteration of refinement

  • solutions – The FEM solution at each iteration

  • est_global_errors – The estimated global error at each iteration

  • ex_global_errors – The exact global error at each iteration

  • N_elements – The size of the mesh

  • color – Whether a color scheme should be used plotting, defaults to True
