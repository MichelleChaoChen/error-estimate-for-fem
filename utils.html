Utilities¶

utils.energy(sol, mesh, a_k, freq, bc_1)¶

Computes the energy norm of the solution.

Parameters:

  • sol – Domain of function

  • mesh – Mesh used for FEM solution

  • a_k – Coefficients of source function

  • freq – Frequency of source function

  • bc_1 – Boundary condition

Returns:

Energy norm on the solution

utils.error_master(xi, sol, mesh, a_k, freq, bc_1)¶

Creates the function for evaluating the energy norm.

Parameters:

  • xi – Domain of function

  • sol – FEM solution

  • mesh – Mesh used for FEM solution

  • a_k – Coefficients of source function

  • freq – Frequency of source function

  • bc_1 – Boundary condition

Returns:

Function for evaluating energy norm

utils.exact_gradient(x, a_k, freq, bc_1)¶

Computes the exact gradient of the solution.

Parameters:

  • x – Domain of function

  • a_k – Coefficients of source function

  • freq – Frequency of source function

  • bc_1 – Boundary condition

Returns:

Exact gradient of solution

utils.exact_sol(x, a_k, freq, bc_1)¶

Computse the exact solution.

Parameters:

  • x – Domain of function

  • a_k – Coefficients of source function

  • freq – Frequency of source function

  • bc_1 – Boundary condition

Returns:

Value of exact solution

utils.f_str(coeff_range, freq_range, N)¶

Function that creates random source function using sines.

Parameters:

  • coeff_range – Range of coefficients of sines

  • freq_range – Range of frequency of sines

  • N – The number of terms in the sum of sines

Returns:

Source function in string format

utils.process_amr_data(filename)¶

Reads data from AMR data file back and returns all data in a dictionary format.

Parameters:

filename – The name of file to read from

Returns:

Data from AMR and average number of iterations per AMR run

utils.write_amr_data(filename, nr_elements, est_global_errors, ex_global_errors)¶

Writes data from AMR to file so that it can be processed later for plotting performance.

Parameters:

  • filename – The name of file to write to

  • nr_elements – Array containing the mesh size

  • est_global_errors – Estimated global errors

  • ex_global_errors – Exact global errors
